# Gift Exchange application

GiftExchange is command-line application to speed up the process to organizes a gift exchange .

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

There are 4 python script.
1.	custom_exception.py
2.	driver.py
3.	gift_exchange.py
4.	test_gift_exchange.py

place all 4 script in a folder and execute below command

```
python driver.py
```
it prompts to enter name and partner() .
Enter member and partner(partner is optional)
once all members are entered , enter new line to run the script that generated the exchange list


### Prerequisites

It requires python 3.2 or higher
